# Front Desafio Toolbox

Este front fue desarrollado con react y bootstrap.

## Para correr el aplicativo es necesario ejecutar el comando:

# `npm start`


### Importante

- El front corre en el puerto 3000, por defecto. en caso de que esté ocupado la app preguntará si desea ejecutar en otro puerto, diga Y en su prompt.
- En caso de no haber conexión al servicio por más de 5 segundos o bien el servicio no está levantado arrojará un error.

### Implementación.

El aplicativo utliza bootrap para el estilo y axios para realizar las llamadas al servicio.
Existe un componente llamado GridComponent que es quien contiene todo el funcionamiento.
